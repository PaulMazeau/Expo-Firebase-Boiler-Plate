---

## Installation

Clonez le dépôt :

```bash
git clone https://github.com/mon-projet/exemple-doc.git
cd exemple-doc
````

Installez les dépendances :

```bash
npm install
```

---

## Utilisation

Lancez le projet :

```bash
npm run dev
```

Puis ouvrez votre navigateur à l'adresse : [http://localhost:3000](http://localhost:3000)

---

## API

### `GET /api/status`

Retourne l’état du serveur :

```json
{
  "status": "ok",
  "uptime": 10234
}
